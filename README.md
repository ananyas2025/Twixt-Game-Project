# Twixt-Game-Project
A simple, CLI version in C of the game - Twixt.

---
## Phase 1 Features:
- Print the game board
- Alternates between the red and black players
- Take input from user in the form of coordinates and place the corresponding peg (red or black) at that positions
- Prints an update version of the board after each move
- Some validity checks (e.g. the spot is not already occupied by another peg, should not place pegs in other colour's boundary region)
